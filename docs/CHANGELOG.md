# Changelog

All notable changes to the Council Snowball Simulator.

## [2.0.0] - The Golden Record
### Added
*   **Unified CLI:** `snowball.py` replaces `main.py` with `play`, `verify`, `chronicle` commands.
*   **Documentation:** `ARCHITECTURE_V2.md` defining the 4-Layer Jurisdiction Graph.
*   **Season 1 Chronicle:** Full compilation of seeds 777-1003.
*   **Soul Protection:** `.gitignore` and `.gitattributes` to protect proprietary character data.

### Changed
*   **Architecture:** Formalized Ontological Layer (Level 0) overriding Audit Layer (Level 3).
*   **Narrative:** Director now supports "Cinematic Bundling" for Audit events.
*   **Core:** Renamed `main.py` to `main.py.bak`.

## [1.5.0] - The Audit
### Added
*   `AuditManager`: Mega's ability to "stabilize" or "ratify" ricochets.
*   `ScenarioManager`: Support for scripted scenarios (`ricochet_audit`).

## [1.4.0] - Bankai & States
### Added
*   `BankaiManager`: Support for "Ultimate States" (Ace's Focus, Mega's Dodge).
*   Jogress Fusion Logic.

## [1.3.0] - The Witness
### Added
*   `auto_narrative_generator.py`: V2 Narrative Engine with Pattern Detection.
*   `ZoomDirector`: Variable pacing for story generation.

## [1.0.0] - The Foundation
### Added
*   Basic Simulation Engine (`simulation.py`).
*   Signal Weavers V1 Logic.
*   Skeletor Ledger (JSONL Logging).
